# Telco Customer Churn Analysis

## Overview
This project analyzes customer churn using the Telco dataset, combining
business insights and financial impact evaluation.

## Key Questions
- What factors drive customer churn?
- Which customers are most at risk?
- What is the financial impact of churn?
- How can the business reduce churn?

## Data
- Source: IBM Telco Customer Churn dataset
- Unit of analysis: Customer
- Key variables: contract type, tenure, payment method, monthly charges

## Methodology
- Data cleaning and preparation
- Exploratory data analysis (EDA)
- Churn segmentation analysis
- Revenue impact estimation

## Outputs
- Business analysis notebook
- Financial analysis notebook
- Key insights and recommendations

## Repository Structure
data/        # Dataset  
notebooks/   # Analysis notebooks (business + financial)  

## Key Insights
- High churn among month-to-month customers  
- New customers are the most vulnerable  
- Certain payment methods increase churn risk  
- Churn represents a direct revenue loss  

## Conclusion
Customer churn is both a behavioral and financial problem.
Reducing churn improves revenue stability and long-term growth.
